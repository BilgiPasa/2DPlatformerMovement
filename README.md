# 2DPlatformerMovement

Scriptlerin üst satırlarında yazmış olduğum yorumları okuyunuz. O yorumlar, Unity editörünüzde birkaç şey yapmanızı isteyebilir.

Projemin .unitypackage dosyası, Unity'nin 2022.3.62f2 versiyonunda Built-In Render Pipeline ile kullanılabilir.

Projemin kodunu kendim yazdım. "Vibe coding" **yapmadım**. Ancak bazı kısımlarda, internette bulduğum rehberlerden yardım aldığım oldu.
